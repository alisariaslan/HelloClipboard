Reporting a Vulnerability

If you discover a security vulnerability in HelloClipboard, please follow these steps:

Contact
Email security reports to: dev@alisariaslan.com
Please do not open a public issue for security vulnerabilities.

Provide Details
Include:

Version of HelloClipboard

Steps to reproduce the vulnerability

Any screenshots, logs, or proof-of-concept code (if possible)

Response Time

We will acknowledge your report within 48 hours.

We aim to provide a patch or mitigation within 14 days depending on severity.

Disclosure Policy

Vulnerabilities will remain confidential until a fix is released.

Once fixed, a public advisory will be published in the repository's Releases and Security Advisories sections.

Security Best Practices

Always use the latest supported version of HelloClipboard.

Avoid running unknown binaries or scripts alongside HelloClipboard.

Review app permissions during installation and configuration.
